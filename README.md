2020年买的游戏，2024年玩，可惜我来的太晚<br>
<br>
见过花就好了，又何必在意花属于谁。花开是有情，花落是无意。<br>
你记得花，花就不会枯萎；你记得我，我就一直在。<br>
似水流年，太过匆匆，一些故事来不及开始，就被写成了昨天；一些人还没好好相爱，就成了过客。离开你的人，是缘分到了，才与你告别。<br>
也许每场相遇，并非都会有结果，但每场相遇，又都会有它的意义。人生，没有无用的经历。你我都要相信，意难平终将和解，万事也终究如意。<br>
或许每个人的青春都有遗憾吧——马不停蹄的暗恋，匆匆忙忙的喜欢，不太理想的成绩，或者在一起却没有长久的他。<br>
时间终会磨平一切，当你真正打开回忆的枷锁，他依旧会出现在你的脑海里，波涛汹涌，久久不能平息。<br>
<br>
其实，我更喜欢好多年前的自己。<br>
他比我有胆量，比我遗憾少，比我懂的少，比我相信的更多。<br>
在失去的所有人中，我更怀念的是我自己。“春风若有怜花意，可否许我再少年！”<br>
<br>
“其实哪有那么多以后？”<br>
就像操场边那棵老槐树——春天不开花，就只能等明年了。<br>
与其在秋天遗憾错过的花期，不如在风起时先把花苞捧在手心。<br>
那些没说出口的话、没敢迈出的步，若总等“合适的时机”，<br>
最后只会变成抽屉里那张写满日期却从未寄出的明信片。<br>
<br>
晚风裹挟着蒲公英的绒毛掠过走廊，<br>
我把笔记本塞进书包，脚步比往常轻快了些——毕竟，最好的档期从来都是现在。<br>
